# MyFitnessRoutines
Android app to create fitness routines.

## OOD
* Routine
  * Name
  * Duration {get return Steps.Duration+Rest}
  * Steps
  * Notification chime
* Step
  * Move
  * Duration
  * Rest Duration
* Move
  * Name
  * Category
  * Singular or L&R
  * Image
  * Description
  * Pose1
  * Pose2
* Pose
  * Torso
  * rLeg
  * lLeg
  * rArm
  * lArm
  * Prop
* Log
  * Session
    * Date
    * Routine Name
    * Duration
