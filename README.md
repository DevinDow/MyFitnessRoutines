# MyFitnessRoutines
Android app to create fitness routines.

## OOD
* Routine
  * Name
  * Duration {get return Steps.Duration+Rest}
  * Steps
  * Notification chime
* Step
  * Pose
  * Duration
  * Rest Duration
* Pose
  * Name
  * Category
  * Singular or L&R
  * Image
  * Description
  * Torso, rLeg, lLeg, rArm, lArm, Prop
* Log
  * Session
    * Date
    * Routine Name
    * Duration
