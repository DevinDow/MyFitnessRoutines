# MyFitnessRoutines
Android app to create fitness routines.

## OOD
* Routine
  * Name
  * Duration {get return Steps.Duration+Rest}
  * Steps
  * Notification chime or watch
* Step
  * Pose
  * Duration
  * Rest Duration
* Pose
  * Name
  * Category
    * Stretch
    * Yoga
    * Lifting
    * Cardio
  * Singular or L&R
  * Image
  * Description
* Log
  * Session
    * Date
    * Routine Name
    * Duration
